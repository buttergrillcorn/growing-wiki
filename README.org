* Description

This is my personal notes wiki, using the working in public / digital garden methodology.

Built with:
- Org mode for writing
- org-id for unbreakable links
- ox-hugo for export
- Hugo + Little Ocean theme for publishing

** Navigation

- Press =SPC W f= to find or create a note
- Press =SPC W s= to search all notes
- Press =SPC W l= to insert a link
- Press =SPC W b= to see backlinks

** Workflow

1. Write notes in Org mode
2. Save automatically exports to Hugo via ox-hugo
3. Git commits and pushes automatically (when enabled with =SPC W a=)
4. Website updates on Cloudflare Pages
