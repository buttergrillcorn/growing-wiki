    &lt;h2 id="description"&gt;Description&lt;/h2&gt;
&lt;p&gt;This is my personal notes wiki, using the working in public / digital garden methodology.&lt;/p&gt;
&lt;p&gt;Built with:&lt;/p&gt;
&lt;ul&gt;
&lt;li&gt;Org mode for writing&lt;/li&gt;
&lt;li&gt;org-id for unbreakable links&lt;/li&gt;
&lt;li&gt;ox-hugo for export&lt;/li&gt;
&lt;li&gt;Hugo + Little Ocean theme for publishing&lt;/li&gt;
&lt;/ul&gt;
&lt;h3 id="navigation"&gt;Navigation&lt;/h3&gt;
&lt;ul&gt;
&lt;li&gt;Press &lt;code&gt;SPC n n&lt;/code&gt; to find or create a note&lt;/li&gt;
&lt;li&gt;Press &lt;code&gt;SPC n G&lt;/code&gt; to search all notes&lt;/li&gt;
&lt;li&gt;Press &lt;code&gt;SPC n I&lt;/code&gt; to insert a link&lt;/li&gt;
&lt;li&gt;Press &lt;code&gt;SPC n b&lt;/code&gt; to see backlinks&lt;/li&gt;
&lt;/ul&gt;
&lt;h3 id="workflow"&gt;Workflow&lt;/h3&gt;
&lt;ol&gt;
&lt;li&gt;Write notes in Org mode&lt;/li&gt;
&lt;li&gt;Save automatically exports to Hugo via ox-hugo&lt;/li&gt;
&lt;li&gt;Git commits and pushes automatically (enabled by default)&lt;/li&gt;
&lt;li&gt;Website updates on Cloudflare Pages&lt;/li&gt;
&lt;/ol&gt;
&lt;p&gt;Toggle auto-sync with &lt;code&gt;SPC n A&lt;/code&gt; if you prefer manual control.&lt;/p&gt;
